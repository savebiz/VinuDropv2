# Audio Assets Required

Please place the following .mp3 files in this directory (`public/sounds/`):

1.  **drop.mp3** - Sound played when an orb is dropped.
2.  **merge.mp3** - Sound played when orbs merge.
3.  **gameover.mp3** - Sound played when the game ends.
4.  **best_score.mp3** - Sound played when a new high score is reached.

Ensure the filenames match exactly.
